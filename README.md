## Setup

> <br/>[!NOTE]<br/> **Ensure Node.js is version <i>^18.18.0 || >=20.10.0.</i>**<br /><br />

1. Install the dependencies.

```bash
# npm
npm install
```

2. Run the project.

```bash
# npm
npm run dev
```

or

## Setup with Docker

```sh
# docker
docker-compose up -d
```
